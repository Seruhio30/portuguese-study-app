
# App Repaso de Portugués



Aplicación web personal para reforzar el aprendizaje de portugués (A1 a B2), enfocada en práctica constante, repaso rápido, vocabulario y seguimiento del progreso.



## Tecnologías

- Frontend: HTML5, CSS3, JavaScript

- Backend: Java + Spring Boot

- Base de datos: MySQL



## Estructura del proyecto

- `backend/`: API y lógica de negocio

- `frontend/`: interfaz web

- `database/`: scripts SQL

- `docs/`: documentación técnica



## Estado

Proyecto en fase inicial.

Mi MVP v1 será una app web que permita:

1. seleccionar un nivel inicial de portugués (A1 o A2)
2. estudiar vocabulario básico según el nivel
3. estudiar verbos frecuentes según el nivel
4. ver conjugaciones de verbos en presente y pretérito perfeito
5. resolver ejercicios escritos simples de conjugación
6. recibir retroalimentación inmediata sobre si la respuesta es correcta o incorrecta
7. guardar progreso básico del usuario en verbos y prácticas realizadas

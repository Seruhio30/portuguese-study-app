# App Repaso de Portugués

## Objetivo
Aplicación web personal para reforzar el aprendizaje de portugués (A1 a B2), enfocada en práctica constante, repaso rápido, vocabulario y seguimiento del progreso.

## Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Java + Spring Boot
- Base de datos: MySQL

## Meta de aprendizaje
Construir el proyecto de forma profesional, paso a paso, para practicar diseño, backend, frontend, base de datos y preparación para entrevistas.

## MVP inicial
- Ver lista de verbos
- Ver conjugaciones básicas
- Resolver ejercicio simple
- Guardar progreso básico

## Regla de trabajo
Cada sesión resuelve solo un bloque.
